# learn-r
R language learning notes.

这个系列的内容使用的文字编辑软件是`RStudio`的notebook模式，可以直接根据R-markdown格式的源文件生成HTML网页，所以可以方便的在浏览器中查看到渲染后的页面，看着也很整洁。

建议下载本仓库后，直接点击对应的HTML文件，浏览顺序按照数字顺序即可☺️。
